BETTERFINGERS (SIGIL SCRIPT)

// INSTALLATION

Requires Python 3.10+ and Rust toolchain.

git clone https://github.com/sourcearcanum/betterfingers.git
cd betterfingers
cargo build --release

// FEATURES

- Whisper Local: Uses quantized Whisper models running on CPU/GPU.
- Macro Weaving: Bind spoken phrases to key sequences in `config.yaml`.
- Vocal Lock: Requires a specific pitch variance to active (anti-accidental trigger).

PDF FORGE (CODEX)

// APPROACH

Most PDF editors upload your document to a cloud server to process it. This is unacceptable for legal or private documents. PDF Forge uses WebAssembly compiled from Ledger libraries to manipulate the byte stream directly in memory.

// ROADMAP

Current version supports split, merge, and rotate. Next version (v0.5) adds cryptographic signing using local PGP keys.

INFINITE AGES (AETHER)

// ECOSYSTEM

Infinite Ages is a JSON-based standard for RPG worlds. It separates "Rules" (systems) from "Lore" (content). You can hot-swap rulebooks while keeping your world history intact.

// GETTING STARTED

Download the Core JSON schema from the repo. Load into any compatible VTT or the official "Aether Viewer" (coming soon).
